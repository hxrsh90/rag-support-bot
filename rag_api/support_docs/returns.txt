We accept returns within 10 days of delivery.

Eligibility:
- Item must be unused and in original condition.
- Tags must be intact.
- Item must be returned in original packaging.

Return Process:
1. Email support@pawcouture.co with your order number.
2. We’ll send a return label.
3. Drop off at any local courier point.

Refunds are processed within 5–7 working days after we receive the return.
